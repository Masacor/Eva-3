// Pasos a seguir //


/Instala Capacitor Android
1.- npm i @capacitor/android@latest

/Crea la carpeta "www", y abre el android studio
/En el celular pide activar la depuración USB
2.- ionic cap build android

/Copia los archivos del proyecto web al proyecto android
3.- npx cap copy android

/Sincroniza y actualiza el proyecto con Capacitor, 
/preparando las carpetas para su ejecución
4.- npx cap sync android

/Ejecuta el proyecto en el celular
5.- npx cap run android
